# 历史粘贴板

一个简洁的 Windows 桌面剪贴板历史记录工具。自动监听剪贴板，保存文字和图片，支持搜索、置顶、一键复制。

## 功能特性

- **自动记录** - 自动监听剪贴板，保存复制的文字和图片，自动置顶到桌面条
- **历史搜索** - 快速搜索历史文字内容
- **一键复制** - 点击历史记录即可再次复制到剪贴板
- **桌面置顶条** - 置顶内容显示在桌面顶部蓝色椭圆条，可拖动、点击展开
- **循环粘贴** - 按快捷键自动将上一条内容粘贴到当前窗口
- **可配置保存时长** - 支持 1/3/5 天自动清理过期记录
- **可配置置顶条数** - 支持 1-50 任意条数
- **开机自启动** - 可在菜单中开关
- **系统托盘** - 最小化到托盘，后台运行
- **全局快捷键** - 显示/隐藏窗口、循环粘贴上一条

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+V` | 显示/隐藏主窗口 |
| `Ctrl+Shift+Q` | 粘贴上一条（每按一次取更早一条，自动粘贴到当前窗口，按键可自定义） |

## 技术栈

- Electron 27 — 桌面应用框架
- React 18 — UI 框架
- Vite 5 — 构建工具
- Tailwind CSS 3 — 样式框架

## 开发

```bash
# 安装依赖
npm install

# 启动开发模式
npm run dev

# 构建前端
npm run build

# 打包 Electron 应用（生成安装包）
npm run build:electron
```

## 项目结构

```
clipboard-history/
├── src/
│   ├── main/          # Electron 主进程（剪贴板监听、数据存储、窗口管理）
│   ├── preload/       # 预加载脚本（IPC 桥接）
│   └── renderer/      # React 渲染进程（UI 界面）
├── docs/              # 项目文档
├── assets/            # 图标等静态资源
└── data/              # 运行时数据（自动创建，不随仓库发布）
```

## 使用说明

1. 启动后应用会在后台监听剪贴板
2. 复制文字或图片后会自动保存并置顶到历史记录
3. 桌面顶部会显示一个蓝色椭圆条，点击可展开查看置顶内容
4. 主窗口可通过托盘图标或 `Ctrl+Shift+V` 呼出
5. 在菜单"设置"中可配置保存时长、置顶条数、快捷键等
6. 关闭主窗口不会退出应用，通过托盘菜单"退出"或菜单栏"文件→退出"退出
