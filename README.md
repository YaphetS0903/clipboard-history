# 历史粘贴板

一个简洁的 Windows 桌面剪贴板历史记录工具。

## 功能特性

- **自动记录** - 自动监听剪贴板，保存复制的文字和图片
- **历史搜索** - 快速搜索历史文字内容
- **一键复制** - 点击历史记录即可再次复制
- **桌面置顶条** - 置顶内容显示在桌面顶部蓝色椭圆条，可拖动、点击展开
- **可配置保存时长** - 支持 1/3/5 天自动清理
- **可配置置顶条数** - 支持 1-50 任意条数
- **开机自启动** - 可在菜单中开关
- **系统托盘** - 最小化到托盘，后台运行
- **全局快捷键** - 
  - `Ctrl+Shift+V` - 显示/隐藏主窗口
  - `Ctrl+Shift+Q`（可自定义）- 循环粘贴上一条

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+V` | 显示/隐藏主窗口 |
| `Ctrl+Shift+Q` | 粘贴上一条（每按一次取更早一条，可自定义按键） |

## 技术栈

- Electron - 桌面应用框架
- React - UI 框架
- Vite - 构建工具
- Tailwind CSS - 样式框架

## 开发

```bash
# 安装依赖
npm install

# 启动开发模式
npm run dev

# 构建
npm run build

# 打包 Electron 应用
npm run build:electron
```

## 使用说明

1. 启动后应用会在后台监听剪贴板
2. 复制文字或图片后会自动保存到历史记录
3. 桌面顶部会显示一个蓝色椭圆条，点击可展开查看置顶内容
4. 主窗口可通过托盘图标或快捷键呼出
5. 在菜单"设置"中可配置保存时长、置顶条数、快捷键等
