# 需求文档 - 历史粘贴板

## 功能需求

### 1. 核心功能

#### 1.1 剪贴板监听
- **自动监听**：后台持续监听系统剪贴板变化
- **内容类型**：
  - 纯文本
  - 富文本（可选）
  - 图片（PNG/JPG/BMP等格式）
- **去重机制**：连续相同内容不重复记录

#### 1.2 数据存储
- **存储位置**：本地文件夹 `data/` 下
- **文字存储**：SQLite 数据库
- **图片存储**：原图保存到 `data/images/`
- **存储期限**：可设置 1天 / 3天 / 5天，到期自动清理

#### 1.3 历史查看
- **排序方式**：时间降序（最新在前）
- **展示形式**：卡片式布局
- **卡片内容**：
  - 文字：前100字预览
  - 图片：缩略图（最大200x200）
  - 时间戳

#### 1.4 卡片操作
- **点击复制**：点击卡片内容重新复制到剪贴板
- **置顶**：将卡片固定在桌面顶部，可部分隐藏，点击展开
- **删除**：单条删除

#### 1.5 搜索功能
- **搜索范围**：文字内容
- **实时搜索**：输入即搜索

### 2. 系统功能

#### 2.1 开机自启动
- 注册 Windows 启动项
- 可选项：用户可关闭

#### 2.2 快捷键
- **呼出界面**：Ctrl + Shift + V
- **粘贴上一条**：Ctrl + Shift + `（反引号）

#### 2.3 托盘图标
- 最小化到系统托盘
- 右键菜单：打开 / 设置 / 退出

## UI设计规范

### 主色调
- 主色：淡蓝色 `#E3F2FD`
- 辅色：天蓝 `#90CAF9`
- 强调：深蓝 `#1976D2`
- 背景：白色 `#FFFFFF`
- 文字：深灰 `#333333`

### 界面布局
- 主窗口：宽度400px，高度自适应屏幕
- 卡片：圆角8px，阴影轻微
- 字体：系统默认微软雅黑

## 非功能需求

### 性能
- 启动时间 < 3秒
- 剪贴板响应延迟 < 100ms
- 支持存储 1000+ 条记录不卡顿

### 安全
- 数据仅本地存储，不上传
- 图片文件权限控制

### 兼容性
- Windows 10/11
- 支持多显示器
