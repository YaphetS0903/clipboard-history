# 开发路线图

## 项目阶段划分

### 阶段一：项目基础搭建（第1天）
**目标**：搭建 Electron + React + Tailwind 基础框架

**任务清单**：
- [ ] 初始化 npm 项目
- [ ] 安装 Electron、React、Vite、Tailwind
- [ ] 配置项目目录结构
- [ ] 配置开发脚本（dev、build）
- [ ] 验证基础窗口能正常启动

**验收标准**：
- `npm run dev` 能启动 Electron 窗口
- 窗口显示 "Hello World" React 组件
- Tailwind 样式生效

---

### 阶段二：数据库与存储（第2天）
**目标**：实现 SQLite 数据库连接和基础 CRUD

**任务清单**：
- [ ] 安装 better-sqlite3
- [ ] 设计数据库表结构（clipboard_items 表）
- [ ] 实现数据库初始化
- [ ] 实现新增记录方法
- [ ] 实现查询记录方法
- [ ] 实现删除记录方法
- [ ] 创建数据存储目录

**验收标准**：
- 数据库文件自动创建在 data/ 目录
- 能通过代码增删改查记录
- 重启应用数据不丢失

---

### 阶段三：剪贴板监听（第3天）
**目标**：实现后台监听剪贴板并保存

**任务清单**：
- [ ] 安装 clipboard-event
- [ ] 实现文字监听和保存
- [ ] 实现图片监听和保存（原图存文件）
- [ ] 实现去重逻辑
- [ ] 测试连续复制同一内容不重复记录

**验收标准**：
- 复制文字自动保存到数据库
- 复制图片保存原图到 data/images/
- 连续复制相同内容只记录一次

---

### 阶段四：主界面开发（第4-5天）
**目标**：实现卡片列表展示

**任务清单**：
- [ ] 设计 Card 组件（文字+图片）
- [ ] 实现 CardList 组件（时间降序）
- [ ] 实现点击复制功能
- [ ] 实现删除功能
- [ ] 实现缩略图显示
- [ ] 添加空状态提示

**验收标准**：
- 主窗口显示历史记录卡片
- 最新内容在最上方
- 点击卡片可复制内容
- 删除后实时刷新列表

---

### 阶段五：搜索功能（第6天）
**目标**：实现历史记录搜索

**任务清单**：
- [ ] 实现 SearchBar 组件
- [ ] 实现搜索 API（支持文字模糊搜索）
- [ ] 搜索结果实时显示
- [ ] 搜索为空时显示提示

**验收标准**：
- 输入关键词实时过滤
- 搜索结果正确
- 清空搜索显示全部

---

### 阶段六：置顶功能（第7-8天）
**目标**：实现卡片置顶到桌面

**任务清单**：
- [ ] 创建置顶窗口（BrowserWindow）
- [ ] 实现置顶窗口收起/展开动画
- [ ] 实现置顶状态管理
- [ ] 实现取消置顶
- [ ] 置顶窗口位置记忆

**验收标准**：
- 点击置顶按钮弹出置顶窗口
- 窗口显示在屏幕顶部
- 可收起/展开
- 重启后置顶状态保留

---

### 阶段七：系统功能（第9天）
**目标**：托盘、快捷键、自启动

**任务清单**：
- [ ] 实现系统托盘图标
- [ ] 实现托盘右键菜单
- [ ] 注册全局快捷键（Ctrl+Shift+V）
- [ ] 实现快捷键呼出/隐藏窗口
- [ ] 实现开机自启动选项

**验收标准**：
- 最小化到托盘

---

### 阶段八：存储期限与清理（第10天）
**目标**：实现自动清理过期数据

**任务清单**：
- [ ] 实现设置界面（存储期限选择）
- [ ] 实现定时清理任务
- [ ] 实现手动清理按钮
- [ ] 清理时同时删除图片文件

**验收标准**：
- 可设置 1/3/5 天存储期限
- 过期数据自动删除
- 关联图片文件同步删除

---

### 阶段九：优化与打包（第11-12天）
**目标**：性能优化和安装包制作

**任务清单**：
- [ ] 优化大图片加载性能
- [ ] 优化大数据量列表性能
- [ ] 添加错误处理
- [ ] 配置 electron-builder
- [ ] 生成 Windows 安装包

**验收标准**：
- 1000+ 条记录不卡顿
- 安装包能正常安装运行
- 卸载后无残留
